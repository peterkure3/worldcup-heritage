# Overall Development Workflow Context

This workspace implements a complete software development pipeline using folder-based orchestration.

**Stage Flow**:
1. 01_planning      → Requirements & planning
2. 02_research      → Technical research & options
3. 03_architecture  → Design & architecture
4. 04_implementation→ Coding & building
5. 05_testing       → Testing & validation
6. 06_review        → Code review & documentation
7. 07_deployment    → Deployment & handover

**How to Use**:
- Start at the appropriate stage based on your current needs.
- Always review and optionally edit output/ from the previous stage.
- Customize stages (add/remove/reorder) as needed.
